# GusIndexer

GusIndexer is a lightweight Python search engine indexer that works seamlessly with [GusCrawler](https://github.com/VERSE-Web/GusCrawler). It converts URLs and web content into a fast, local, and searchable index.

## Features

- Index URLs from JSON outputs of GusCrawler
- Fast full-text search using Whoosh
- Local, offline searchable index
- Easy to extend for more advanced features like page titles and content

## Requirements

- Python 3.8+
- Whoosh (`pip install whoosh`)
